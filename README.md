# README

# 🍞アプリ概要
パン好きな人が自分の好きなパンを投稿できるアプリ

# 💻開発環境

- Ruby 2.6.5
- Ruby on Rails 6.0.0
- MySQL 0.5.3 (DB)
- Github
- Visual Studio Code

# 🥖実装機能
- ログイン機能
- 投稿機能
- 画像ファイルのアップロード機能
- 投稿へのコメント機能
- 投稿の、詳細表示、編集、削除の3機能
- ページネーション
- 都道府県の選択機能

# 🥐使用したGem
- gem 'pry-rails'
- gem 'devise'
- gem 'carrierwave'
- gem 'mini_magick'
- gem 'kaminari'
- gem 'active_hash'
