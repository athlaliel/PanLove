# README

# アプリ概要
パン好きな人が自分の好きなパンを投稿できるアプリ

# 開発環境

- Ruby 2.6.5
- Ruby on Rails 6.0.0
- MySQL 0.5.3 (DB)
- Github
- Visual Studio Code

# 実装機能
- ログイン機能
- 投稿機能(文章、写真)
  ※編集、削除は投稿者のみ可能
- 投稿のコメント機能
- ページネーション
